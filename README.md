1.	Duty Check: A Teacher logs in and sees the list of assigned Invigilation Assignments.
2.	Access Attendance Sheet: On the day of the exam, the Teacher views the digital Attendance Module for their assigned hall. This view is pre-populated with the names and seat numbers of all students allocated to that hall.
3.	Mark Attendance: The Teacher marks each student as Present (P) or Absent (A). This action updates the Attendance Record model in real-time.
4.	Reporting: Once the exam is complete, the Teacher can export the attendance sheet as a signed CSV report for official submission
